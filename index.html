WizarDroid

A lightweight Open Source Android library for creating step by step wizards

View project onGitHub

WizarDroid is a lightweight Open Source Android library developed by CodePond.org that addresses a feature that Android is surprisingly missing, Wizards. If you reached this page, rest assured THERE IS NO BUILT-IN WIZARD SUPPORT IN ANDROID (as of writing this page). Learn more.

Usage Advantages

  • Wizard's flow is defined in one place and can be maintained easily
  • Step context is persistent and passed between steps automatically using reflection
  • Simple and consistent API for controlling wizard's flow in runtime

Technical Overview

WizarDroid is built on top of Android's Fragments and is compatible with Android API 9+ using Android Support Library. Learn more.
