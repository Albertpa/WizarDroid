Detailed Description

The engine of the Wizard. This class controls the flow of the wizard and is using ViewPager under the hood. You would normally want to extend WizardFragment instead of using this class directly and make calls to the wizard API via org.codepond.wizardroid.WizardFragment#wizard field. Use this class only if you wish to create a custom WizardFragment to control the wizard.
